Welcome to Rudder

Description
Manage your IT infrastructure by making inventories of your nodes, and applying Rules to them.
A Rule is the application of a configured Technique to a node group.

Rudder Asset Manager
  • Accept or refuse new nodes in Rudder
  • List or search your nodes' inventories in Rudder
  • Define node groups from search results
Rudder Support Desk
  • Report and follow bugs, join the Rudder community!
  • Professional Services and plugins for your organization
Rudder Configuration Manager
  • Build your own Technique library and create and configure Directives instances
  • List, register and manage Rules on your nodes from available Directives
  • Manage and supervise states from these Rules
Rudder Administration
  • Track all events that have occured: Event logs
  • Configure the Rudder server
