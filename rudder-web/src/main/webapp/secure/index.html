Welcome to Rudder

Description
Rudder allows you to manage your IT infrastructure by making an inventory of your nodes, and applying configuration rules to them.
A configuration rule is the application of a configured policy template to a node group.

Rudder Asset Manager
  • Accept or refuse new nodes in Rudder
  • List or search your nodes' inventories in Rudder
  • Define node groups from search results
Rudder Support Desk
  • Report and follow bugs, join the Rudder community!
  • Professional Services and plugins for your organization
Rudder Configuration Manager
  • Build your own policy template library and create and configure instances of policies
  • List, register and manage configuration rules on your nodes from available policies
  • Manage and supervise states from these configuration rules
Rudder Administration
  • Track all events that have occured: Event logs
  • Configure the Rudder server
