Groups

A group is based on a query over inventory data that defines a set of nodes, on which specific policies can then be applied.

Group categories are there to visually organize groups, and have no effect on the content of the groups or the applied configuration.

There is a set of pre-defined groups (called system groups), that covers Rudder-related classification.
